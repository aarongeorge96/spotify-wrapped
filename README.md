\## How to Use



1\. Clone the repo

2\. Add your Spotify data to `data/` folder

3\. Run the notebook in `notebooks/`



\## Get Your Own Data



1\. Go to https://www.spotify.com/account/privacy/

2\. Request "Download your data"

3\. Wait for email (few days to 30 days)

4\. Replace the CSV in `data/` folder
